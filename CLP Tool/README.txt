The CLP.py is a Command Line Programm that prints the last 5 lines of a file. 

***************************************************************
***                         USAGE                           ***
***************************************************************
To run the CLP simply type the following command in terminal:
     python CLP.py filename

***************************************************************
***                      PRE-REQUISITES                     ***
***************************************************************
#run the following command to install BinaryOrNot Python package 
(checks whether a file is binary or text)
   pip install binaryornot
